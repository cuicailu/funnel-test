# 漏斗分析2.0 自动化测试脚本使用说明

## 环境要求

你的电脑已经安装好以下内容（已确认）：
- Python 3.14
- Playwright
- pytest + pytest-playwright

---

## 文件说明

| 文件名 | 说明 |
|---|---|
| `test_funnel_analysis.py` | 自动化测试脚本主文件 |
| `auth.json` | 登录状态文件（首次运行后自动生成） |

---

## 第一步：把脚本文件放到你的电脑上

1. 在你的电脑上新建一个文件夹，例如：`C:\funnel_test`
2. 把 `test_funnel_analysis.py` 文件放进这个文件夹

---

## 第二步：在命令行中进入该文件夹

打开命令提示符（Win+R 输入 cmd），输入：

```
cd C:\funnel_test
```

---

## 第三步：运行测试

输入以下命令运行所有测试用例：

```
python -m pytest test_funnel_analysis.py -v
```

- `-v` 表示显示详细结果，每条用例的通过/失败都会列出来

---

## 第四步：查看测试报告

如果想生成 HTML 格式的测试报告，运行：

```
python -m pytest test_funnel_analysis.py -v --html=report.html
```

运行完成后，文件夹里会出现 `report.html`，用浏览器打开即可查看。

---

## 重要说明

> 当前脚本中的具体元素定位代码（如 `.funnel-chart-container`）已注释掉，等漏斗分析2.0功能上线后，需要根据实际页面元素补充这些代码。补充方式：把注释掉的 `#` 删除，并根据实际页面调整元素选择器。

---

## 如何更新脚本（功能上线后）

1. 打开 `test_funnel_analysis.py`
2. 找到对应测试函数
3. 删除注释符号 `#`，根据实际页面元素修改选择器
4. 重新运行测试

---

## 常见问题

**Q：运行时提示找不到 pytest？**

A：在命令行输入以下命令后重试：
```
python -m pip install pytest pytest-playwright
```

**Q：运行时浏览器一闪而过看不到操作过程？**

A：默认是无头模式（不显示浏览器）。如果想看到浏览器操作过程，运行：
```
python -m pytest test_funnel_analysis.py -v --headed
```
